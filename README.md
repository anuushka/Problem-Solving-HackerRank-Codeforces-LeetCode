Essential problems to add to each branch:

### 1. HackerRank Branch (Focus: **Python Mastery**)
These problems demonstrate that you understand Python’s high-level features and its specific data handling capabilities.

* **"The Minion Game":** (Strings/Logic) – Great for showing efficient string slicing and memory management without over-complicating loops.
* **"Compress the String!":** (Itertools) – Perfect for demonstrating your knowledge of `itertools.groupby`, which is a very "Pythonic" way to handle data.
* **"Validating Credit Card Numbers":** (Regex) – Shows you can handle data validation using Regular Expressions, a highly sought-after skill in business development and strategy.

### 2. LeetCode Branch (Focus: **Interview Patterns**)
These are the "Big 3" patterns that appear in almost every technical interview at companies like Amazon or Google.

* **"Number of Islands":** (Graphs/BFS/DFS) – The gold standard for showing you can traverse a 2D grid.
* **"Longest Palindromic Substring":** (Dynamic Programming) – Demonstrates that you can optimize an $O(n^3)$ problem down to $O(n^2)$ using DP.
* **"Merge k Sorted Lists":** (Linked Lists/Heaps) – Uses a **Priority Queue** (Heap) to solve a complex sorting problem. It’s a classic "Hard" problem that impresses reviewers.

### 3. Codeforces Branch (Focus: **C++ & Optimization**)
These test your ability to handle massive constraints and advanced graph theory.

* **"Road Construction" (Problem 25C):** (Graphs/Kruskal/DSU) – A perfect application for your **DSU** (Disjoint Set Union) interest to maintain connected components.
* **"Shortest Routes I & II":** (Dijkstra/Floyd-Warshall) – These are essential for showing you can implement shortest-path algorithms on graphs with up to $10^5$ nodes.
* **"String Problem" (Problem 33C):** (Prefix Sums/Optimization) – A great way to show how you can optimize a $O(n^2)$ logic into a linear $O(n)$ solution.
